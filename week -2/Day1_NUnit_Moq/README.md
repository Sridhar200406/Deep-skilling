# Week 2 - Day 1: Entity Framework Core Basics

## Objectives
- Install EF Core packages
- Create DbContext
- Create Entity Models
- Run Initial Migration
- Perform CRUD Operations

## Assignment
Build a simple Student Management System using EF Core.

### Tasks
1. Create Student entity.
2. Create AppDbContext.
3. Configure SQL Server connection.
4. Create migration.
5. Insert sample students.
6. Display all students.
7. Update a student.
8. Delete a student.
